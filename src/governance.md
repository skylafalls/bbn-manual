# The BBN Foundation - Goverance

This file defines the governance structure for the BBN Foundation pseudo-organization.

## Definitions

- The key words “MUST”, “MUST NOT”, “REQUIRED”, “SHALL”, “SHALL NOT”, “SHOULD”, “SHOULD NOT”, “RECOMMENDED”, “MAY”, and “OPTIONAL” in this document are to be interpreted as described in [RFC 2119](https://www.rfc-editor.org/rfc/rfc2119).
- AC: Administration Council.
- FG: Fictional Googology.
- MFGC: Monthly Fictional Googology Conference meeting.
- FGFC: The Fictional Googology Fanclub, always refers to Skylafalls's Fictional Googology Fanclub except otherwise specified.
- FGWC: FG Wiki Committee.
- rFGC: Rigorous FG Council.
- iFGC: Informal FG Council.
